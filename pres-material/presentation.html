What is Docker ?

Background

  • A company (french [!] located in california): Docker, Inc
  • CEO : Solomon Hykes
  • Which releases an Open source product : Docker
  • Since 2013
  • Written in Go lang.
  • Github : 30k commits, 38k stars, 11k forks.
Linux kernel : 40k stars

Main Purpose

Pack, ship and run any application as containers

What is a container

Docker containers wrap up a piece of software in a complete filesystem that contains everything it needs to run: code, runtime, system tools, system libraries & anything you can install on a server.
